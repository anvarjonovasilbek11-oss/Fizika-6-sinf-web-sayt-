# Fizika 6 - Interaktiv Ta'lim Platformasi 🚀

O'zbekiston maktablarining 6-sinf o'quvchilari uchun maxsus ishlab chiqilgan, zamonaviy va interaktiv fizika o'rganish platformasi.

## ✨ Imkoniyatlar

- **📚 To'liq Darslik**: 1-bobdan 7-bobgacha bo'lgan barcha mavzular, nazariya va tajribalar.
- **🤖 AI Test Generator**: Sun'iy intellekt yordamida har bir mavzu bo'yicha maxsus testlar yaratish.
- **🎥 Video Darslar**: Mavzularni tushunishni osonlashtiruvchi interaktiv video darslar to'plami.
- **📂 O'quv Qo'llanmalar**: PDF va boshqa formatdagi qo'llanmalarni saqlash va yuklab olish imkoniyati.
- **🌓 Dark Mode**: Ko'zni charchatmaydigan premium qora rejim (doimiy ravishda faol).
- **🌐 Ko'p tilli interfeys**: O'zbek, Rus va Ingliz tillarida to'liq qo'llab-quvvatlash.

## 🛠 Texnologiyalar

- **Frontend**: React.js, Vite
- **Styling**: Tailwind CSS, Framer Motion (premium animatsiyalar)
- **Icons**: React Icons (Remix Icon)
- **Storage**: LocalForage (IndexedDB) - oflayn ma'lumotlarni saqlash uchun
- **AI Integration**: Groq API (Llama 3) testlarni generatsiya qilish uchun

## 🚀 Ishga tushirish

1. Repo-ni klonlang:
   ```bash
   git clone https://github.com/anvarjonovasilbek11-oss/Fizika-6-sinf-web-sayt-.git
   ```
2. Bog'liqliklarni o'rnating:
   ```bash
   npm install
   ```
3. `.env` faylini yarating va API kalitni qo'shing:
   ```env
   VITE_GROQ_API_KEY=your_api_key_here
   ```
4. Loyihani ishga tushiring:
   ```bash
   npm run dev
   ```

## 👨‍💻 Admin va O'quvchi rejimlar

- **O'quvchi**: Testlarni yechish, darsliklarni ko'rish va materiallarni yuklab olish imkoniyatiga ega.
- **Admin**: Yangi testlar yaratish, ularni tasdiqlash va materiallarni boshqarish huquqiga ega.
